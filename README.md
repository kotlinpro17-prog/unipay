# UniPay – University Payment System

UniPay is a graduation project designed to simplify university payments for students.

## Overview
This system allows students to pay their university fees online بسهولة وأمان.

## Key Features
- Secure login with OTP verification via SMS
- Payment confirmation messages
- Notifications and reminders for due payments
- User-friendly interface for students

## SMS Usage
SMS will be used for:
- Sending OTP verification codes
- Payment confirmations
- Alerts and reminders

## Project Status
Currently under development as part of a university graduation project.

## Purpose
This project is intended for academic use and aims to solve real problems in university payment systems.
